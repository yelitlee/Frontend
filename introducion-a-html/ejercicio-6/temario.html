Curso de desarrollo Frontend

Un desarrollador Front End debe tener conocimiento sobre HTML, CSS y JavaScript y ser capaz de utilizar algunos frameworks o librerías para crear todo tipo de interfaces de usuarios. React, Redux, Angular, Bootstrap, Foundation, LESS, Sass, Stylus y PostCSS son algunos de ellos.
